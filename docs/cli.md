# xp CLI

The `xp` CLI keeps common project tasks in one place.

| Command | Purpose |
| --- | --- |
| `xp create my-api` | Create a fresh app. |
| `xp run` | Configure, build, and run the current app. |
| `xp build` | Build without starting the server. |
| `xp watch` | Watch source files and rebuild on changes. |
| `xp clean` | Remove the local build folder. |
| `xp doctor` | Check basic tool availability. |

## Start a project

```bash
xp create my-api
cd my-api
xp run
```
